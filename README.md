# Temporizador

Aplicativo desktop criado com o objetivo de se ter um temporizador.

Faz uma contagem regressiva de 1 hora e ao acabar o tempo manda uma notificação ao usuário.

## Tecnologias 
- [x] HTML
- [x] CSS
- [x] JavaScript
